# 📊 Store Sales Analysis (2022)

## 📌 Project Overview
This project involves a comprehensive analysis of retail store sales data for the year 2022. The objective was to clean, transform, and visualize the data to identify key business insights and growth opportunities.

## 🛠️ Skills Used
- **Data Cleaning:** Removing duplicates, handling null values, and standardizing categories.
- **Data Transformation:** Creating new columns like 'Age Group' and 'Month' for better segmentation.
- **Pivot Tables:** Aggregating data to find trends.
- **Data Visualization:** Building an interactive dashboard in MS Excel.

## 🔍 Key Insights
- **Top Customers:** Adult women (25-50 years) are the primary buyers (~70%).
- **Best Month:** March recorded the highest sales.
- **Top Channels:** Amazon, Flipkart, and Myntra drive the most volume.
- **Geography:** Bengaluru, Hyderabad, and New Delhi are the top-performing cities.

## 📂 Files Included
- `Store_Sales_Analysis.xlsx`: The main Excel file with Data & Dashboard.
- `Project_Presentation.pdf`: A detailed slide-based report.
- `Screenshots/`: Visuals of the dashboard.

## 💡 Recommendation
Focus marketing spend on **Adult Women** via **Amazon**, specifically in top-tier cities like **Bengaluru and Hyderabad** to maximize ROI in 2023.
